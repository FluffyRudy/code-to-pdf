Relational Algebra and SQL Queries

I. What are the names of students enrolled in cs3020?

SQL:

SELECT name 
FROM student 
JOIN enrolledIn ON student.id = enrolledIn.id 
WHERE enrolledIn.code = 'cs3020';

II. Which subjects is Hector taking?

SQL:

SELECT subject.code 
FROM subject 
JOIN enrolledIn ON subject.code = enrolledIn.code 
JOIN student ON student.id = enrolledIn.id 
WHERE student.name = 'Hector';

III. Who teaches cs1500?

SQL:

SELECT lecturer 
FROM subject 
WHERE code = 'cs1500';

IV. Who teaches cs1500 or cs3020?

SQL:

SELECT lecturer 
FROM subject 
WHERE code = 'cs1500' 
   OR code = 'cs3020';

V. Who teaches at least two different subjects?

SQL:

SELECT lecturer 
FROM subject 
GROUP BY lecturer 
HAVING COUNT(DISTINCT code) >= 2;

VI. What are the names of students in cs1500 or cs3010?

SQL:

SELECT name 
FROM student 
JOIN enrolledIn ON student.id = enrolledIn.id 
WHERE enrolledIn.code = 'cs1500' 
   OR enrolledIn.code = 'cs3010';

VII. What are the names of students in both cs1500 and cs1200?

SQL:

SELECT name 
FROM student 
WHERE id IN (SELECT id 
             FROM enrolledIn 
             WHERE code = 'cs1500') 
   AND id IN (SELECT id 
              FROM enrolledIn 
              WHERE code = 'cs1200');

VIII. What are the names of students in at least two different subjects?

SQL:

SELECT name 
FROM student 
WHERE id IN (SELECT id 
             FROM enrolledIn 
             GROUP BY id 
             HAVING COUNT(DISTINCT code) >= 2);

IX. What are the codes of all the subjects taught?

SQL:

SELECT DISTINCT code 
FROM subject;

X. What are the names of all the students?

SQL:

SELECT name 
FROM student;

XI. What are the names of all the students in cs1500?

SQL:

SELECT name 
FROM student 
JOIN enrolledIn ON student.id = enrolledIn.id 
WHERE enrolledIn.code = 'cs1500';

XII. What are the names of students taking a subject taught by Roger?

SQL:

SELECT name 
FROM student 
JOIN enrolledIn ON student.id = enrolledIn.id 
JOIN subject ON enrolledIn.code = subject.code 
WHERE lecturer = 'Roger';

XIII. What are the names of students who are taking a subject not taught by Roger?

SQL:

SELECT name 
FROM student 
JOIN enrolledIn ON student.id = enrolledIn.id 
JOIN subject ON enrolledIn.code = subject.code 
WHERE subject.code NOT IN (SELECT code 
                           FROM subject 
                           WHERE lecturer = 'Roger');
